¿Que es el DOM?

DOM significa Document Object Model, en español sería Modelo de Objetos del Documento. Es una interfaz de programación que nos permite crear, cambiar, o remover elementos del documento También podemos agregar eventos a esos elementos para hacer más dinámica nuestra página.

Nuestro documento se llama nodo raíz y contiene un nodo hijo el cual es el elemento html. El elemento html contiene dos hijos los cuales son los elementos head y body.

El DOM visualiza el documento de HTML como un árbol de tres nodos. Un nodo representa un documento de HTML.

¿Es util utilizar el DOM?

El DOM (Document Object Model) es extremadamente útil y esencial en el desarrollo web! Es la interfaz de programación que permite a los desarrolladores interactuar con los documentos HTML y XML de una manera estructurada y programática.

¿Por qué es útil utilizar el DOM?

  1. Interactividad dinámica

    • El DOM te permite hacer que una página web sea interactiva y responda a las acciones del usuario. Puedes agregar, eliminar o modificar elementos en la página en respuesta a eventos como clics, desplazamientos o entradas del usuario.
    • Ejemplo: Mostrar u ocultar un menú cuando el usuario hace clic en un botón.

  2. Manipulación de Contenidos

    • Puedes modificar el contenido de un documento sin tener que recargar la página. Esto es clave para aplicaciones web dinámicas, donde los datos se actualizan en tiempo real.
    • Ejemplo: Cambiar el texto de un párrafo o el valor de un campo de formulario en función de una acción del usuario.

  3. Modificación de la Estructura del Documento

    • Puedes añadir o eliminar elementos HTML (como div, p, img, etc.) de la página dinámicamente. Esto te permite crear interfaces más complejas y personalizadas.
    • Ejemplo: Crear un nuevo elemento de lista y agregarlo a un menú desplegable basado en una acción del usuario.

  4. Acceso y Manipulación de Estilos (CSS)

    • El DOM también te permite acceder y modificar los estilos de los elementos de la página, lo que te brinda la capacidad de cambiar la apariencia de los elementos de manera dinámica.
    • Ejemplo: Cambiar el color de un botón cuando el usuario pasa el mouse por encima o cambiar la visibilidad de un elemento.

  5. Eventos y Control de Flujo

    • Puedes capturar y manejar eventos del usuario (como clics, teclas presionadas, movimientos del mouse, etc.) a través del DOM. Esto es fundamental para crear experiencias de usuario interactivas y dinámicas.
    • Ejemplo: Ejecutar una función cuando se envía un formulario o se selecciona un ítem de un menú.

  6. Aplicaciones Web Modernas

    • El DOM es la base para tecnologías modernas como React, Vue, y Angular, que dependen del DOM para renderizar dinámicamente los componentes y gestionar los estados de las aplicaciones. Aunque algunas de estas librerías y frameworks abstraen la manipulación directa del DOM, el conocimiento del DOM sigue siendo crucial.

  7. Integración con APIs del Navegador

    • Muchas APIs del navegador están diseñadas para interactuar con el DOM. Por ejemplo, la API de geolocalización puede usarse para mostrar la ubicación del usuario en el DOM, o la API de almacenamiento local puede interactuar con el DOM para mostrar datos almacenados localmente.


Como seleccionar elementos en el documento

Hay diferentes métodos para seleccionar un elemento en el documento de HTML.

De los cuales solo nos enfocaremos en 3 metodos, que son:

  1. getElementById
  2. querySelector
  3. querySelectorAll


1. getElementById

En HTML, los ids se utilizan como identificadores únicos para los elementos HTML. Esto significa que no podemos tener el mismo nombre de id para dos elementos diferentes.
El método getElementById permite, como su nombre indica, seleccionar un elemento del documento por medio del valor del atributo id que se le haya asignado.


2. querySelector

querySelector te permite selecionar elementos del DOM mediante a queries, es decir, si usas . seleccionas por clase, si usas # por id, si usas [] por nombre de atributos, y así sucesivamente, puedes buscar los diferentes selectores disponibles pero es el equivalente al $ de jQuery


3. querySelectorAll

Este método encuentra todos los elementos que coinciden con el selector de CSS y devuelve una lista de todos esos nodos. Si quisiera encontrar todos los elementos en nuestro ejemplo, podría utilizar el combinador de hijos > para encontrar a todos los elementos hijos de ul.
